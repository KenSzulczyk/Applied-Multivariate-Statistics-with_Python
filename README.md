# Applied Multivariate Statistics with Python

This repository contains all Python code, datasets, and supporting materials for the textbook *Applied Multivariate Statistics with Python*.

## 📘 Textbook

The full textbook is available in the `textbook/` folder.

> **Note:** Copying code directly from the PDF may introduce formatting issues (e.g., spacing and line breaks). For best results, use the scripts or notebooks provided in this repository.

## 📂 Repository Structure

* `data/` – Datasets used in examples and exercises
* `figures/` – Images and visualizations from the book
* `notebooks/` – Jupyter notebooks organized by chapter
* `scripts/` – Clean Python scripts for key methods and models
* `textbook/` – Full textbook in PDF format

## 🚀 Getting Started

1. Clone or download this repository
2. Install required Python packages (see `requirements.txt` if available)
3. Open the notebooks in Jupyter or Google Colab
4. Follow along with the corresponding chapter in the textbook

## 🎯 Purpose

This repository is designed to:

* Help students reproduce results from the book
* Provide hands-on experience with multivariate statistical methods
* Serve as a practical companion to the textbook

## 📊 Topics Covered

* Principal Component Analysis (PCA)
* Factor Analysis
* Cluster Analysis
* Discriminant Analysis
* Other multivariate techniques
